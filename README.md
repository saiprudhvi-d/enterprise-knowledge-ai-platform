# enterprise-knowledge-ai-platform
End-to-end enterprise RAG platform using LangChain + OpenAI + Pinecone for natural language querying across 500K+ documents. Multi-step agentic workflows for retrieval, summarization, and Q&amp;A. Stack: Python, LangChain, OpenAI API, Pinecone, FastAPI, Docker, Kubernetes, AWS ECS, CI/CD
